橫跨多種平台

速度與效能

透過 Web Worker 和伺服器端渲染，達到在如今(以及未來）的 Web 平台上所能達到的最高速度。

Angular 讓你有效掌控可伸縮性。基於 RxJS、Immutable.js 和其它推送模型，能適應海量資料需求。

美妙的工具

使用簡單的宣告式樣板，快速實現各種屬性。使用自定義元件和大量現有元件，擴充套件樣板語言。在幾乎所有的 IDE 中獲得針對 Angular 的即時幫助和反饋。所有這一切，都是為了幫助你編寫漂亮的應用，而不是絞盡腦汁的讓程式碼『能用』。
